To add border to the button just call the class "border_to_color" eg (class='sato border_to_red')


To change the color of any button just call up the class "to_color" eg (class ='sato to_chocolate')


To add shadow just call the class "shadow" eg (class= 'sato shadow')


To make the button smaller use the class "xtra_small" eg (class= 'sato xtra_small')


To make the size medium call the class "medium" eg (class= 'sato medium')


To make the size large call the class "large" eg (class= 'sato large')


To get your desired style and size call the classes in this form(class= 'sato sa_side to_chocolate border_to_black shadow')
